## PHASE 1 KURZÜBERSICHT — AUFBAU & ERÖFFNUNG

**Ziel:** Vom leeren Raum zum eröffneten Café.

| Meilenstein | Wann | Abhängigkeit |
|---|---|---|
| Nutzungsänderung Bauamt eingereicht | Jetzt | — |
| Gesundheitsamt Voranmeldung | Parallel | — |
| Elektriker + Klempner beauftragt | Nach Genehmigung | Bauamt |
| Geräte bestellt (gebraucht) | Parallel recherchieren | — |
| Personal rekrutiert | 6 Wochen vor Öffnung | — |
| Lieferanten fixiert (Bäcker, Kaffee, Großmarkt) | 4 Wochen vor Öffnung | — |
| Soft-Launch (geschlossener Kreis) | 2 Wochen vor Öffnung | Alles oben |
| Öffentliche Eröffnung | Q3/Q4 2026 | Soft-Launch OK |

→ Details: `docs/AMUN_CAFE_AUFBAU_EROEFFNUNG.md`

## PHASE 2 KURZÜBERSICHT — LAUFENDER BETRIEB

**Ziel:** Jeden Tag profitabel, effizient und ohne Chaos.

| Bereich | Kernfrage | Antwort |
|---|---|---|
| Tagesablauf | Was passiert wann? | 6:00 Mise-en-place, 7:00 Öffnung, 18:00 Abschluss |
| Einkauf | Was, wo, wie oft? | Täglich Bäcker, 2x/Woche Großmarkt, monatlich Spezialitäten |
| Finanzen | Wie viel müssen wir machen? | ~80–120 Transaktionen/Tag für Break-Even |
| KPIs | Wie merken wir, ob es läuft? | Food Waste %, Umsatz/Tag, Wartezeit, Google-Bewertung |
| Food Waste | Wie vermeiden? | Tagesmengen protokollieren, KI-Vorhersage ab Monat 3 |

→ Details: `docs/AMUN_CAFE_BETRIEBSHANDBUCH.md`

## PHASE 3 KURZÜBERSICHT — GÄSTEBETREUUNG

**Ziel:** Jeder Gast geht zufrieden — und kommt wieder.

| Kanal | Tool | Standard |
|---|---|---|
| Im Café | Persönlicher Service | Wartezeit < 3 Min., Name merken, Stammkunden erkennen |
| Website/WhatsApp | KI-Chatbot | 24/7 verfügbar, Allergen-HITL, Fallback ans Team |
| Google | Bewertungsmanagement | Jede Bewertung beantwortet, ≤ 24h |
| Instagram | Content + Reaktion | Täglich Stories, Kommentare beantworten |
| Beschwerden | Sofort-Protokoll | Entschuldigung → Lösung → Follow-up |

→ Details: `docs/AMUN_CAFE_GASTEBETREUUNG.md`
