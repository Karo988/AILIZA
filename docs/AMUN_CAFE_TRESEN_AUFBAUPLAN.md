SANDWICH-WORKFLOW (Zero-Waste-Motion):

  SCHRITT 1          SCHRITT 2           SCHRITT 3

  (links)        →   im Mittelgang   →   (rechts)

## THERMO- & SICHERHEITS-CHECK

### Hitzequellen vs. Kältezonen:

```
HITZEQUELLEN:              KÄLTEZONEN:
──────────────             ──────────────
Kontaktgrill 4,4 kW   ←──── 1,20 m ABSTAND ────► Kühltisch Sandwich
Espressomaschine 3,5kW ←──── RÜCKWAND trennt ────► Kuchenvitrine
Spülmaschine 2,9 kW   ←──── NASSZONE isoliert───► Eis-Vitrine Emma

ERGEBNIS: KEINE direkte thermische Übertragung zwischen Hitze und Kälte ✅
```

### Sicherheitsempfehlungen:

| Punkt | Problem | Lösung |
|---|---|---|
| **Starkstromanschluss** | 12–13 kW Gesamtlast | Elektriker: 3-Phasen-Anschluss prüfen |
| **Kühlaggregat-Freiräume** | Emma + Kühltisch brauchen Luftzirkulation | mind. 5 cm Wandabstand seitlich |
| **Kontaktgrill (4,4 kW)** | Hitzeentwicklung nach oben | Kein Holzregal direkt darüber |
| **Nassbereich Spüle** | Spritzwasser neben Elektrogeräten | Spritzschutz-Trennwand 30 cm |
| **Handwaschbecken** | Gesundheitsamt Pflicht | Separat, nicht die Spüle! |

## INSTALLATIONSREIHENFOLGE (Aufbauplan)

```
WOCHE 1 — VOR GERÄTELIEFERUNG:
  ✓ Elektriker: alle Anschlüsse legen (Schienen unter Arbeitsplatten)
  ✓ Klempner: Wasseranschluss Spüle + Spülmaschine + Handwaschbecken
  ✓ Boden versiegeln / Fliesen im Thekenbereich

WOCHE 2 — UNTERBAU:
  ✓ Kühltheke Unterbau einbauen (Kühltisch links)
  ✓ Spülmaschine Unterbau einsetzen
  ✓ Mülltrennsystem einbauen
  ✓ Arbeitsplatten auflegen + verschrauben

WOCHE 3 — GROSSE GERÄTE:
  ✓ Eis-Vitrine Emma anliefern + positionieren (schwer! 2 Personen)
  ✓ Kuchenvitrine aufstellen + anschließen
  ✓ Spüle einbauen + anschließen

WOCHE 4 — KLEINE GERÄTE + FINISH:
  ✓ Espressomaschine + Mühle aufstellen
  ✓ Kontaktgrill / Toaster aufstellen
  ✓ Slush-Maschine aufstellen
  ✓ Kassensystem einrichten (TSE aktivieren!)
  ✓ Alle Geräte Probelauf
  ✓ Gesundheitsamt-Begehung vorbereiten
```
