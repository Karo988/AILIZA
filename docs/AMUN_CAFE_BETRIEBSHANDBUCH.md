## 4. FINANZEN & BREAK-EVEN

### Fixkosten pro Monat (Schätzung):

| Position | Betrag/Monat |
|---|---|
| Miete (geschätzt, abhängig vom Vertrag) | 800–1.500 € |
| Energie (Strom, Heizung) | 300–500 € |
| Personal (1 Vollzeit oder 2 Teilzeit) | 1.800–2.800 € |
| Versicherungen (Betrieb, Haftpflicht) | 150–250 € |
| Buchhaltung / Steuerberater | 100–200 € |
| Software (Kasse, Buchhaltung) | 50–100 € |
| Marketing (laufend) | 100–200 € |
| **Fixkosten GESAMT** | **~3.300–5.550 €** |

### Variable Kosten (Wareneinsatz):
- **Ziel:** Wareneinsatz max. 28–33% vom Umsatz
- Bei 9 € Sandwich: max. 2,70–3,00 € Wareneinsatz pro Sandwich
- Bei 4 € Kaffee: max. 1,20–1,30 € Wareneinsatz

### Break-Even-Rechnung:

```
SZENARIO: 4.500 € Fixkosten/Monat

Durchschnittlicher Bon: 7,50 € (Kaffee + kleines Essen)
Wareneinsatz: 30% = 2,25 €
Deckungsbeitrag pro Bon: 5,25 €

Break-Even = Fixkosten / Deckungsbeitrag pro Bon
           = 4.500 € / 5,25 €
           = ~857 Bons/Monat
           = ~29 Bons/Tag (26 Arbeitstage)

Realistisches Tagesziel: 60–100 Bons/Tag
Bei 80 Bons/Tag: ~42.000 € Umsatz/Monat (vor Steuern)
Gewinn nach Fixkosten + Wareneinsatz: ~8.000–12.000 €/Monat
```

**Fazit:** Ab 30 Transaktionen pro Tag bist du nicht mehr in den Miesen. Ab 80 ist es ein gutes Geschäft.
